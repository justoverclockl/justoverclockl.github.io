# COMPETENZE

Ho terminato da poco un corso Full Stack durato 7 mesi, durante i quali ho appreso e approfondito i più comuni linguaggi di programmazione quali Javascript e Typescript, ed ho anche appreso come utilizzare il framework React per il frontend e NodeJs per il backend.

Nello specifico qui in basso descrivo alcune delle mie competenze:

Html, Css, Javascript

La Santa Trinità! I fondamentali presenti praticamente in ogni pagina web. La struttura con HTML5, lo stile e le animazioni con CSS3 e la dinamicità e versatilità di Javascript.

ES6, ES7, Typescript

Grazie all'aiuto di Typescript si può finalmente "scrivere" senza errori, e grazie alle funzioni di Es6 ed Es7 finalmente ora è anche più semplice implementare funzioni spettacolari all'interno dei siti web.

React

React e le SPA (Single Page Application), sono stati amore a prima vista, sviluppare applicazioni dinamiche che non necessitano di ricaricare la pagina per visualizzare i dati modificati mi ha decisamente incuriosito, spingendomi ad approfondirlo ed impararlo.

NodeJs

Poter dare voce a Javascript lato server, è sicuramente un punto fondamentale nello sviluppo di applicazioni e siti web, per questo ho deciso di apprendere questo potentissimo framework in modo da poter applicare le conoscenze javascript anche lato backend.
